# 💡 Что объединяет мои проекты

Каждый следующий проект добавляет новые знания и демонстрирует освоение новых технологий.

Например:

- после изучения PyQt5 я разработала приложения с графическим интерфейсом;
- затем научилась работать с REST API и JSON;
- освоила использование современных AI-моделей через OpenRouter API;
- научилась проектировать более сложную архитектуру приложений и разделять их на независимые модули.

Таким образом каждое приложение является небольшим завершённым этапом обучения.
